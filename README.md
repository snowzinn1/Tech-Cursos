# 🚀 Tech-Cursos

Plataforma web desenvolvida para visualizar cursos da área de tecnologia.
O projeto foi desenvolvido utilizando **React com Vite** no frontend e uma **API mockada com JSON Server** para simular um backend.

A aplicação permite consumir dados de cursos através de uma API e exibir essas informações de forma dinâmica na interface.

---

# 🧑‍💻 Tecnologias utilizadas

### Frontend

* React
* Vite
* JavaScript
* HTML
* CSS

### Backend (API mockada)

* JSON Server
* db.json

### Deploy

* Vercel (Frontend)
* Render (API)

# ⚙️ Como executar o projeto localmente

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/snowzinn1/Tech-Cursos.git
```

### 2️⃣ Acessar a pasta do projeto

```bash
cd Tech-Cursos
```

### 3️⃣ Instalar as dependências

```bash
npm install
```

### 4️⃣ Rodar o projeto

```bash
npm run dev
```

O projeto irá iniciar em:

```
http://localhost:5173
```

---

# 🔌 Rodando a API mockada

O projeto utiliza uma API simulada através do **JSON Server** utilizando o arquivo `db.json`.

Para rodar a API localmente:

```bash
npx json-server --watch db.json --port 3000
```

A API ficará disponível em:

```
http://localhost:3000
```
